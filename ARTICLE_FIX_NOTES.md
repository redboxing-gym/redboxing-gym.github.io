# 아티클 수정 메모

생성된 아티클마다 추후 수정하고 싶은 내용을 기록하는 문서입니다.  
작업 전에는 이 파일을 먼저 확인하고, 수정이 끝난 항목은 완료 표시와 날짜를 남깁니다.

## 기록 방식

- 상태: `대기`, `진행중`, `완료`, `보류`
- 우선순위: `높음`, `보통`, `낮음`
- 가능한 한 어떤 문단, 제목, CTA, 이미지, 링크를 바꾸고 싶은지 구체적으로 적습니다.

예시:

```md
### 수정 항목
- 상태: 대기
- 우선순위: 보통
- 요청일: 2026-05-28
- 내용: 본문 두 번째 소제목에 행신점 위치 설명을 더 자연스럽게 추가
```
